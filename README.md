                                                                                CYBER-RANGER

Cyber Safety Awareness and Reporting Initiative for Goa

CYBER-RANGER is a dedicated, community-driven web platform focused on combating rising cyber threats in Goa. We provide local, actionable information on common scams (like UPI fraud, fake job offers, and digital arrests) and offer a clear, immediate pathway for citizens to report and recover from incidents.

This project is built to promote digital literacy and aligns with SDG 4: Quality Education by providing practical safety guidance to students, seniors, professionals, and small businesses across the state.

The CYBER-RANGER platform is designed to promote Awareness, Prevention, and Recovery through the following features:

    Awareness First: Clear, plain-language explanations of threats relevant to Goans, such as Phishing, Ransomware, UPI/QR Frauds, Digital Arrests, and Social Engineering.

    Action-Oriented Prevention: Simple Do/Don't checklists and advice on safe habits (e.g., verifying payments, device hygiene).

    Immediate Reporting: Prominent guidance on the Report & Recover flow, emphasizing the critical need to call 1930 immediately and file a complaint on cybercrime.gov.in.

    Cyber Ambassador Mobilization: A contact form to easily request and engage with volunteers for local workshops and sessions.

    Resource Hub: Direct links and simple walkthroughs for essential tools like Sanchar Saathi (for blocking lost devices and SIM verification).

    Trust and Accessibility: Clean layout, high-contrast palette, clear visual hierarchy, and focus states for accessible keyboard navigation.

This project is built as a multi-page, responsive web application using fundamental frontend technologies:

    Languages: HTML, CSS, JavaScript

    Structure: Semantic HTML5 structure.

    Styling: Organized CSS (styles/main.css and styles/responsive.css) utilizing CSS Grid/Flex for responsive layouts.

    Interactivity: JavaScript (scripts/main.js) handles client-side features like smooth scrolling and simple card animations via IntersectionObserver.
